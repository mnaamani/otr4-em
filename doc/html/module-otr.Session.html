Class: Session

otr. Session

Constructor

new Session(user, account, contact, parametersopt)

Session class is an EventEmitter, that handles the OTR protocol. You can optionally add an online() method to the instance which should return true if contact is online. This determines wether to send hearbeats or not. If method is not added contact is assumed to be online.
